Установка и настройка пакета

Установка

Пакет программ обработки данных UPRISE работает в операционной системе Microsoft Windows.

Для установки пакета UPRISE достаточно распаковать архив с программой на жёсткий компьютера. Абсолютный путь установки не важен. Возможно использование (в том числе и одновременное) нескольких копий пакета.

Кроме программ и вспомогательных файлов пакет, входящий в установочный архив, для работы пакета UPRISE требуется наличие библиотек автокорреляционных функций (АКФ). Библиотеки АКФ могут устанавливаться на любой диск, имеющий достаточно свободного пространства (около 70 Гбайт). Для разных копий пакета UPRISE достаточно одной копии библиотек АКФ. Путь к ним задаётся при настройке пакета и может быть любым. Но, тем не менее, не рекомендуется использовать пути, содержащие не латинские буквы и пробелы.

Настройка

Пакет программ UPRISE настраивается путём редактирования содержимого конфигурационного файла config.dat. Это текстовый файл может быть открыт любым редактором, но рекомендуется ассоциировать расширение dat с тексовым редактором, имеющим возможность нумерации строк (например, notepad++). Вызов редактора после ассоциирования расширения может производиться из оболочки пакета - программы UPRISE.exe.

в n) n - номер строки данного файла

1) файл характеристики разрядника

4) длительность зондирующего импульса
5) минимальная высота
6) максимальная высота
7) шаг по высоте
8) значение гелия на первой высоте
9) значение гелия на других высотах
10) модули градиентов + и - по Te
11) модули градиентов + и - по Ti
12) модули градиентов + и - по H+
13) Количество СКО для параметров
14) Ширина окна для расчёта СКО (сейчас должно быть чётным числом)
15) Шаг при 1-м приближении (H+, Ti, Te)
16) Интервалы (+-) при 2-м приближении (H+, Ti, Te)
17) Шаг при 2-м приближении
18) Интервалы (+-) при 3-м приближении
19) Шаг при 3-м приближении
20) Интервалы (+-) при 4-м приближении
21) Шаг при 4-м приближении
22) Шаг по гелию
23) Кислородное приближение (используйте осторожно!!!) 0 - отключено, 1 - включено
