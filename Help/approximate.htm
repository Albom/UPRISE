Высотная коррекция

Высотная коррекция профилей значений КФ НР сигнала необходима из-за пространственного смещения результирующего рассеивающего объёма, возникающего при импульсном режиме работы радара. Существует несколько способов такой коррекции. В пакете UPRISE наиболее эффективные из них реализованы.

Скользящее окно

Для коррекции профилей значений КФ НР сигнала по этому методу используется скользящая полиномиальная аппроксимация. Для всех точек высотного профиля с использованием точек ниже и выше по методу наименьших квадратов независимо друг от друга строятся полиномы. Степень полиномов должна быть невысокой (оптимальным является полином третьей степени). После нахождения коэффициентов полиномов вычисляются значения задержек АКФ на всех высотах.

Для высотной коррекции с применением скользящего окна используется программа 04_UPRISE_approximate_polynomial.exe. Запуск может осуществляться из Оболочки пакета UPRISE.

После запуска программы просмотра необходимо ввести дату начала измерений в формате D M YYYY (например, 28 1 2011) и количество суток измерений, а также параметры процедуры коррекции – номер минимальной высоты, начиная с которой её производить (ниже будет осуществляться интерполирование данных) и ширину окна (его нужно задавать не меньше 4, так как по умолчанию в программе используются полиномы третьей степени).

Данные для коррекции берутся из каталога step2, а результаты работы программы записываются в каталог step3 в каталоге проекта (ДДММГГ-ДДММГГ).

Трапецеидальное суммирование

Для высотной коррекции методом трапецеидального суммирования данных используется программа 04_UPRISE_approximate_trapezoidal.exe. Запуск может осуществляться из Оболочки пакета UPRISE.

После запуска программы просмотра необходимо ввести дату начала измерений в формате D M YYYY (например, 28 1 2011) и количество суток измерений, а также параметр трапецеидального суммирования.

Данные для коррекции берутся из каталога step2, а результаты работы программы записываются в каталог step3 в каталоге проекта (ДДММГГ-ДДММГГ).
