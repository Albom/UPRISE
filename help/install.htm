Установка и настройка пакета

Установка

Пакет программ обработки данных UPRISE работает в операционной системе Microsoft Windows.

Для установки пакета UPRISE достаточно распаковать архив с программой на жёсткий компьютера. Абсолютный путь установки не важен. Возможно использование (в том числе и одновременное) нескольких копий пакета.

Кроме программ и вспомогательных файлов, входящих в установочный архив, для работы пакета UPRISE требуется наличие библиотек автокорреляционных функций (АКФ). Библиотеки АКФ могут устанавливаться на любой диск, имеющий достаточно свободного пространства (около 20 Гбайт). Для разных копий пакета UPRISE достаточно одной копии библиотек АКФ. Путь к ним задаётся при настройке пакета и может быть любым. Но, тем не менее, не рекомендуется использовать пути, содержащие не латинские буквы и пробелы.

Настройка

Пакет программ UPRISE настраивается путём редактирования содержимого конфигурационного файла config.dat. Это текстовый файл может быть открыт любым редактором, но рекомендуется ассоциировать расширение dat с тексовым редактором, имеющим возможность нумерации строк (например, notepad++). Вызов редактора после ассоциирования расширения может производиться из оболочки пакета - программы UPRISE.exe.

Для настройки программ обработки данных, полученных в IV режиме работы харьковского радара, используется конфигурационный файл config_COld.dat.

Для настройки программ обработки данных, полученных по короткому импульсу в 12 режиме работы харьковского радара, используется конфигурационный файл config_short.dat.

Для настройки программы 3_UPRISE_estimate_wave.exe для исследования волновых процессов в ионосфере используется конфигурационный файл config_wave.dat.
